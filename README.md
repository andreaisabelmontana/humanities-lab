# humanities-lab

Interactive reasoning companion to IE BCSAI *Humanities: Technology and the Making of the Modern World* (Patrick De Oliveira). 10 self-contained widgets that let you manipulate the course's central argument — that technology is not a neutral force of progress but a human creation that reorganizes power.

**Live:** https://andreaisabelmontana.github.io/humanities-lab/

Module 1 — Situating the machines (Winner, Benjamin, Misa)
1. Do artifacts have politics? — sort Langdon Winner's cases into flexible / inherently political / neutral, with feedback on each reading
2. Determinism vs. social construction — slide between "technology drives history" and "society shapes technology" and watch the same event get renarrated
3. Anatomy of a scholarly argument — Toulmin model (claim · grounds · warrant · backing · rebuttal) for the course's key theses; toggle a layer off to feel the argument weaken

Module 2 — The long arc, 1740–2016 (Misa, *Leonardo to the Internet*)
4. Periodizing the modern world — drag a lens across the timeline to read each era's driving technology, the order it reshaped, and the matching session
5. Who gains, who loses? — redistribute a technology's benefits across four groups and read the resulting inequality (Gini)

Module 3 — Industry, empire, modernity
6. The factory clock — the Lowell mills: set the working day and loom speed to see hours converted into output, wage, and owner surplus
7. Tools of empire — toggle steamship, railway, telegraph, quinine and the repeating rifle to assemble nineteenth-century imperial capability (reach · speed · force)

Module 4 — Reading and resisting
8. Reading sources critically — classify excerpts as primary or secondary/scholarly, with notes on what each can and cannot tell you
9. Civilization & barbarism — weigh a technology's achievements against its costs on Walter Benjamin's dialectical scale
10. Spotting the determinist fallacy — name the rhetorical move (technological determinism · lone-genius myth · "tech is neutral" · Whiggish progress) that hides technology's politics

Plain HTML + canvas + KaTeX. Indigo accent. Zero build step. Hand-built, with no AI-generated text — in keeping with the course's AI policy.
